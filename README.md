<h1 align="center"> Descrição </h1>

Este projeto visa auxiliar um agricultor local, chamado de Seu Paulo a gerenciar as suas plantações. Esse projeto visa auxiliar ao Seu Paulo a conseguir aumentar a produtividade e a qualidade da colheita.

<h2>Visão Geral do Projeto -</h2>

    O projeto visa resolver o problema crítico do agricultor: Seu Paulo planta suas estações fora da estação.

    Para isso, o software auxilia, avisando-o quando ele tenta realizar um cadastro de uma plantação fora de suas estações recomendadas. Além disso, o software conta com um sistema que ativa aspersores, baseado em horários criados pelos usuários, e conta com o cadastro de sensores, que podem sentir que se há um indice alto ou baixo de luz, se há pouca umidade no ar, ou se está fazendo muito ou pouco calor.

<h2>Escopo </h2>
    O sistema é capaz de fazer:

    <h3>Cadastro de usuario: </h3>
    Gerencia o acesso

    <h3>Cadastro de plantas: </h3>
    Os administradores podem cadastrar para os usuários tipos novos de plantas

    <h3>Gestão de plantações: </h3>
    Cadastro de plantações, criação de alarmes para irrigação, recebe alarmes quando as plantações estiverem prontas para a colheita 

    <h3>Gestão de alarmes: </h3>
    Permite que o usuário crie e exclua alarmes.
